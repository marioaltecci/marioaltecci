# Hi there! I'm Mario Altecci 🔍

I am an independent **Manual QA Specialist** and Open Source enthusiast. I do this for free in my spare time because I believe software should be intuitive, polished, and accessible.

### Why I focus on Manual QA (vs. Automation):
While Python-based automated tests are great for checking if a button "exists," they often fail to see if the app actually **makes sense** to a human.
* **Context Awareness:** I catch logical errors in translations (e.g., when "Behavior" is wrongly translated as "Biometrics").
* **Visual Integrity:** I identify UI alignment issues and text truncations on different screen resolutions.
* **UX/UI Audit:** I evaluate the app from a real user's perspective to ensure the interface is intuitive.

### Important Note on my Activity:
My GitHub contribution graph doesn't show the full picture. 
* **Email Outreach:** I often contact developers directly via **email** with detailed bug reports and feedback, so many of my contributions happen outside of GitHub.
* **Pro Bono:** This is a hobby driven by a desire to improve the apps I use every day.

### My Goals:
* 🛠 **Bug Hunting:** Detailed reports with steps to reproduce.
* 🌐 **Localization:** Ensuring high-quality translations and proper context.
* 📈 **Product Quality:** Helping developers fix those "small" UI bugs that ruin the experience.
