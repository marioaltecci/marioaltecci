# Hi there! I'm Mario Altecci 🔍

I am an independent **Manual QA Specialist** and Open Source enthusiast. I do this pro bono in my spare time because I can't stand broken interfaces and lazy localization. If an app has a "crooked" UI, I'm here to find it.

### Why I choose Manual QA (vs. Automation):
Automation is great for checking if code runs, but it's useless for checking if an app is actually **usable** for a human. 
* **Logic over Scripts:** I find "hidden" bugs that Python scripts miss — like logical errors in translations or buttons that overlap.
* **UX/UI Integrity:** I identify alignment issues, text truncations, and "dirty" layouts on real devices.
* **Human Perspective:** I evaluate the product from a user's point of view to ensure the experience is intuitive and polished.
